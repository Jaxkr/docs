This endpoint allows you to view the project configuration for a particular project you've made. This returns the tag schema, the FPS, the pipeline configuration, and other fields of the project.

## Get Project Info

### Request

```shell Terminal
curl https://api.sievedata.com/v1/get_projects \
    -H "X-API-Key: YOUR_API_KEY"
```


| Property | Type | Description |
| ---- | ----- | ---- |
| `project_name` | string | The name of the project |

### Sample response

```json
{
    "description": "successful",
    "projects": [
        "amazing_project_1",
        "amazing_project_2",
        "amazing_project_x"
    ],
    "customer_info": {
        "API-Key": "YOUR_API_KEY",
        "can_set": True,
        "can_store": True,
        "name": "YOUR_NAME",
        "hours_left": NUM_HOURS_LEFT
    }
}
```
