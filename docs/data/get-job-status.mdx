---
title: "Get Status of Specific Job"
---

<RequestSimple method="GET">https://api.sievedata.com/v1/get_job_status</RequestSimple>

Get the status of a specific job. The status is either `queued`, `processing`, `finished`, or `failed`.

### Parameters

| Parameter | Type | Description |
| ---- | ----- | ---- |
| `project_name` | string | The name of the project |
| `job_id` | string | The job id of a videeo you've submitted |

### Request

```bash Terminal
curl 'https://api.sievedata.com/v1/get_job_status?project_name=amazing_project_1&job_id=add3978a-94ac-4585-ab52-8bc5c2295c6f' \
  -X GET \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Sample Response

```json
{
  "description": "Successful",
  "status": "finished"
}
```
